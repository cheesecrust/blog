React와 TypeScript를 사용하여 간단한 블로그를 만들어 보았습니다.

## 사용한 기술 스택

- **React** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Vite** - 빌드 도구
- **Tailwind CSS** - 스타일링
- **React Router** - 페이지 라우팅

## 구조

페이지는 크게 두 가지로 나뉩니다. 글 목록을 보여주는 메인 페이지와 개별 글을 읽을 수 있는 상세 페이지입니다. Navbar 없이 깔끔하게 글에만 집중할 수 있도록 구성했습니다.
